# Store Item Demand Forecasting

Прогнозирование ежедневного спроса для 500 временных рядов (10 магазинов × 50 товаров) с сравнением линейных, tree-based и статистических подходов.

## Задача

Построить модель прогноза продаж (`sales`) по дате, магазину и товару. Сравнить классические ML-модели с учётом сезонности, тренда и production-сценария (ежедневное дообучение).

## Датасет

- **Источник:** [Store Item Demand Forecasting (Kaggle)](https://www.kaggle.com/c/demand-forecasting-kernels-only)
- **Объём:** 913 000 строк, 3 года ежедневных данных
- **Признаки:** `date`, `store`, `item`, `sales`
- **Паттерны:** восходящий тренд, месячная и недельная сезонность

## Подход

1. **EDA временного ряда** — агрегаты по дате, сезонность, распределение sales
2. **Feature engineering:**
   - Календарные признаки (day_of_week, month, year)
   - Циклическое кодирование (sin/cos для дня недели и месяца)
   - Лаги (1, 7, 14, 30 дней) и скользящие средние / std
   - Экспоненциальное затухание (EWM) и momentum
3. **Линейная модель** — ElasticNet с TimeSeriesSplit, log-трансформация таргета
4. **Tree-based модели** — LightGBM, Random Forest с objective=`mape`
5. **Архитектурный эксперимент** — глобальная vs 500 локальных моделей (store × item)
6. **Production-симуляция** — walk-forward на 92 дня с ежедневным переобучением
7. **SARIMA** — обсуждение ограничений (500 моделей × ~3 мин = непрактично)

## Результаты

| Модель | MAPE | Примечание |
|--------|------|------------|
| Linear Regression | 15.8% | R² = 0.87, хороший baseline |
| 500 локальных LR | 18.0% | Overfitting на локальный шум |
| Random Forest | 13.9% | — |
| **LightGBM** | **12.85%** | objective=mape, без log-трансформации |
| LightGBM (prod walk-forward) | **12.85%** | Стабильно на горизонте 92 дня |

**Ключевые инсайты:**
- Глобальная модель лучше 500 локальных — локальные переобучаются на шум конкретной точки
- LightGBM опирается на rolling mean за 30 дней; EWM-признаки улучшили интерпретируемость
- Prod-симуляция показала стабильность MAPE (12.1% → 14.3% на длинном горизонте)
